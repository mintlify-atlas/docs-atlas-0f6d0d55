The web3Wallet API is a RESTful HTTP API built with NestJS. It accepts and returns JSON, and follows standard HTTP conventions for methods, status codes, and error responses.

## Base URL

```
http://localhost:3000
```

Run the server locally with `npm run start:dev`. The server listens on port 3000 by default, as configured in `src/main.ts`.

## Request format

All request bodies use JSON. Set the `Content-Type` header when sending a body:

```bash
Content-Type: application/json
```

## Response format

Successful responses return JSON with an HTTP 2xx status code. The health endpoint is an exception — it returns a plain string.

```json
{
  "address": "0xAbCd1234...",
  "balance": "1000000000000000000",
  "network": "mainnet"
}
```

Error responses follow a standard NestJS error shape:

```json
{
  "statusCode": 400,
  "message": "Validation failed",
  "error": "Bad Request"
}
```

## HTTP methods

| Method | Usage |
|--------|-------|
| `GET` | Retrieve a resource |
| `POST` | Create a new resource |

## Status codes

| Code | Meaning |
|------|---------|
| `200` | Request succeeded |
| `201` | Resource created |
| `400` | Invalid request body or parameters |
| `404` | Resource not found |
| `500` | Internal server error |
