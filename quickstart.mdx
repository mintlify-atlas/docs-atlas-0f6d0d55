## Prerequisites

- [Node.js](https://nodejs.org) v18 or later
- npm, yarn, or pnpm

## Setup

<Steps>
  <Step title="Clone the repository">
    Clone the web3Wallet repository to your local machine.

    ```bash
    git clone https://github.com/smonderoy/web3Wallet.git
    cd web3Wallet
    ```
  </Step>

  <Step title="Install dependencies">
    Install the project dependencies using your preferred package manager.

    <CodeGroup>

    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```

    ```bash pnpm
    pnpm install
    ```

    </CodeGroup>
  </Step>

  <Step title="Start the development server">
    Run the server in watch mode so it restarts automatically on file changes.

    <CodeGroup>

    ```bash npm
    npm run start:dev
    ```

    ```bash yarn
    yarn start:dev
    ```

    ```bash pnpm
    pnpm run start:dev
    ```

    </CodeGroup>

    <Note>
      The server listens on port **3000** by default, as configured in `src/main.ts`. See the [Configuration](/configuration) page to change the port.
    </Note>
  </Step>

  <Step title="Verify the server is running">
    Send a request to the root endpoint to confirm the API is responding.

    ```bash
    curl http://localhost:3000
    ```

    You should receive the following response:

    ```text
    Hello World!
    ```

    <Tip>
      The `GET /` route is handled by `AppController` and delegates to `AppService.getHello()`, which returns the string `'Hello World!'`.
    </Tip>
  </Step>
</Steps>

## Available scripts

All scripts are defined in `package.json`.

| Script | Command | Description |
|---|---|---|
| `start` | `nest start` | Start the server without watch mode |
| `start:dev` | `nest start --watch` | Start in watch mode (recommended for development) |
| `start:prod` | `node dist/main` | Run the compiled production build |
| `test` | `jest` | Run unit tests |
| `test:e2e` | `jest --config ./test/jest-e2e.json` | Run end-to-end tests |
| `test:cov` | `jest --coverage` | Run tests with coverage report |
