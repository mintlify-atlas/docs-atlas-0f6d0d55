# web3Wallet API

The web3Wallet API is a backend service built with [NestJS](https://nestjs.com), providing a scalable and maintainable foundation for Web3 wallet operations. It handles account management, transaction processing, and blockchain interactions through a clean RESTful interface.

## What is web3Wallet?

web3Wallet is a backend API service designed to power Web3 wallet functionality. Built on NestJS — a progressive Node.js framework — it provides:

- **Modular architecture** — cleanly separated controllers, services, and modules for each domain
- **TypeScript-first** — full type safety across the entire codebase
- **RESTful API** — standard HTTP endpoints for wallet and transaction management
- **Testable by design** — unit and end-to-end tests with Jest and Supertest

## Getting started

<Steps>
  <Step title="Install dependencies">
    Clone the repository and install packages with your preferred package manager.
  </Step>
  <Step title="Configure the environment">
    Set up environment variables for your blockchain network and wallet settings.
  </Step>
  <Step title="Run the server">
    Start the development server and confirm the API is responding on port 3000.
  </Step>
  <Step title="Make your first request">
    Call the health endpoint to verify everything is working correctly.
  </Step>
</Steps>
